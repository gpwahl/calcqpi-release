How to Cite this Project
========================

If you publish work using calcQPI, or code building on algorithms in calcQPI, please cite the paper:

- P. Wahl, L.C. Rhodes, C.A. Marques, *calcQPI: A versatile tool to simulate quasi-particle interference*, [SciPost Physics Codebases 61 (2025)](https://scipost.org/10.21468/SciPostPhysCodeb.61).
- P. Wahl, L.C. Rhodes, C.A. Marques, *Codebase release 1.0 for calcQPI*, [SciPost Physics Codebases 61-r1.0 (2025)](https://scipost.org/10.21468/SciPostPhysCodeb.61-r1.0).
